# yandex_vacancy_search

Автоматизированный поиск вакансий на Яндекс Карьере (портал вакансий компании Яндекс) + фильтрация через LLM под ваш стек.

> Если первичный найм автоматизирован, то и поиск вакансий можно автоматизировать подумал я. Подумал, можно и сделать.

---

## Зачем это

Решил я поискать работу не на ХХ а на сайтах бигтеха. Как же я был наивен...
* потратил несколько часов
* прошерстил сто вакансий (80% которые мне не релевантны (опыт, стек и тп))
* нашел, думаю "вот она, рыба моей мечты". Подался -> автоотказ через 10 минут
* пришло осознание что что-то не так...

Решил сделать инструмент (пока только яндекс):
* 1 кнопка, максимально простой UI
* никаких БД, все происходит в режиме реального времени
* простая архитектура (оч удобно)
* ваши глазки и нервы это теперь головняк "AI vacancy search"

p.s. нет, это не пет-проект. Это инструмент которым я сам пользуюсь. Пользуйтесь и вы.
---

## Что делает конкретно (пайплайн)

1. Парсит вакансии с Яндекса (через внутренний API, не HTML а значит есть пагинация)
2. Учитывает blacklist (вакансии, куда уже откликнулся / отказали)
3. Прогоняет новые вакансии через LLM (API OpenRouter. Он копеечный и за пару баксов может жить месяц)
4. Формирует рекомендации с релевантными вакансиями

---

## Как конкретно работает

* уникальность по `url`
* обновление `matched_keywords` (если вакансия попалась по разным запросам. А в смежных отраслях типо DS/ML такое часто бывает)
* отчет содержит только релевантные вакансии (`score >= 9`. Но учтите что решение о релевантности принимает LLM, а не вы сами)

## Запустить очень просто

```bash
git clone ...
cd project

python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows

pip install -r requirements.txt
```

Настройка:

* `.env`
- OPENROUTER_API_KEY= зарегайтесь на OpenRouter и добавьте ключ. Используйте .env.example для примера
- остальные параметры .env можно не трогать

Запустите:

```bash
python app.py
```

Откройте в браузере на пк:

```
http://127.0.0.1:5000
```

## Использование

В UI всего одна кнопка "Запустить анализ":

### 1. Сначала заполняете поле "Профиль / стек (опишите свой профиль и стек)"

* пишете максимально кто вы, что вы умеете
* можете использовать мой шаблон
* чем конкретнее напишете, тем релевантнее вакансии будут

### 2. Заполняете поле System prompt для LLM

* промт от меня который используется по умолчанию
* можете поменять если результаты вас не устраивают
* надо эксперементировать, идеальных промтов не бывает
* на данный момент самый релевантный под разработчика на питон тот который сейчас

### 3. Заполняете поле Blacklist

* сюда вставляете ссылки, одна ссылка на одной строке
* это ссылки на вакансии на которые вы: откликнулись, нехотите откликатся и т.д.
* эти ссылки парсер просто проигнорирует

### 4. Как все заполнено нажимаете на кнопку "запустить анализ"

* ждете от 1 до 10 минут
* парсит долго, обрабатывает быстрее

### 5. В поле рекомендации появятся подходящие Вам вакансии
